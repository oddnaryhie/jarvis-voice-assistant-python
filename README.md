# Jarvis Voice Assistant

A voice-controlled assistant built using Python.

## Features
- Wake word detection ("Jarvis")
- Speech recognition
- Text-to-speech responses
- Open Google, YouTube, Facebook, and LinkedIn
- Music playback support

## Technologies Used
- Python
- speech_recognition
- pyttsx3
- webbrowser

## Future Improvements
- News API integration
- OpenAI integration
- Weather updates
- Reminder system
